# vLLM настройки
0. Настройки запуска смотри в dockers/vllm
1. ## Основные настройки и параметры
   vLLM поддерживает стандартные параметры генерации OpenAI: 
   Роли (Roles):
   system: Задает общее поведение модели (инструкции).
   user: Сообщение от пользователя.
   assistant: Ответ модели (используется для передачи истории диалога).
   Параметры генерации:
   temperature: Управляет случайностью. 
   0 — детерминированный ответ (факты), 
   1 и выше — креативность.
   top_p: (Nucleus sampling) Выбор токенов, суммарная вероятность которых достигает. Рекомендуется изменять либо temperature, либо top_p, но не оба сразу.
   max_tokens: Лимит на длину генерируемого ответа.
   presence_penalty / frequency_penalty: Штрафы за повторение слов или тем.
   stop: Список строк (стоп-слов), при появлении которых генерация прекращается.
