## МАШИННЫЙ ПЕРЕВОД & ПОИСК И СОЗДАНИЕ ОПИСАНИЙ
1. Для production версии предлагаю встроенный переводчик TranslateGemma от Google:
   1. https://blog.google/innovation-and-ai/technology/developers-tools/translategemma/
   2. 55 языков; 
   3. полностью бесплатный;
   4. без ограничений по количеству запросов (устанавливается локально на своем сервере и не зависит от внешних факторов)
   5. качество перевода обещают высокое;
2. Для нормальной работы требуется:
   1. процессор с поддержкой AVX2 (доп инструкции для параллельных вычислений больших объемов данных - появилась в 2013 году);
   2. и/или видеокарта с хорошим процессором (25-30 тыс.руб.) - наиболее оптимально все вычисления производить на граафическом процессоре; 
   3. в этом случае перевод будет осуществляться за несколько миллисекунд - т.е. практически незаметно - дольше будет идти запись в базу данных;
   4. для 2026 года эти требования наверное ниже средних, но у меня сервер сделан на: 
      1. Intel(R) Core(TM)2 Quad CPU    Q6600  @ 2.40GHz
      2. с видеокартой GeForce 8600 GTS
   5. Этого недостаточно для запуска TranslateGemma, поэтому для разработки будем использовать MyMemory(см. ниже)
   6. Для production version все равно нужно подбирать облачное решение - есть много вариантов с арендой GPU по времени использования
